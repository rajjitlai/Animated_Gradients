# Animated Gradient Text Hero

A visually striking hero section featuring animated gradient text, a moving glowing blob background, and a dark translucent overlay for a modern, immersive feel.

![Animated Gradient Text Hero Preview](/assets/preview.png)

## Features
- **Animated Gradient Text**: Smoothly cycling color gradients applied to the hero typography.
- **Dynamic Background**: A glowing, blurred orb that moves smoothly across the screen.
- **Dark Theme**: A translucent overlay that provides depth and contrast while maintaining the underlying background colors.
- **Modern UI**: Clean typography and interactive elements.

## Tech Stack
- HTML5
- CSS3 (Vanilla)
- Google Fonts (Poppins)
